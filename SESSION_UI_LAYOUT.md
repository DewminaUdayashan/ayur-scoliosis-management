## Positioning Logic

```
Screen Dimensions: width ร height

Video Call Indicator (Bottom-Right):
  x = width - 16 - 150    // 16px margin, 150px width
  y = height - bottomPadding - 70 - 56

Session Notes Indicator (Bottom-Left):
  x = 16                  // 16px margin from left
  y = height - bottomPadding - 70 - 56

Both at same vertical position but opposite sides
```
